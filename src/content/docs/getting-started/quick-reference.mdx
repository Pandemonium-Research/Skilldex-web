# Quick reference

The most common `skillpm` commands at a glance.

## Install a skill

```bash
skillpm install <skill-name>
skillpm install <skill-name> --scope project
skillpm install <skill-name>@1.0.0
```

## Search the registry

```bash
skillpm search <query>
```

## List installed skills

```bash
skillpm list
skillpm list --scope project
skillpm list --scope global
```

## Uninstall a skill

```bash
skillpm uninstall <skill-name>
```

## Validate a skill

```bash
skillpm validate ./path/to/skill
```

## Publish a skill

```bash
skillpm publish
```

## Get help

```bash
skillpm --help
skillpm <command> --help
```

## Check version

```bash
skillpm --version
```
